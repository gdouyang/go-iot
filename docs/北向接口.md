#### 设备开关
`/north/control/{deviceId}/switch` 
- 请求内容
```
[{index:0,status:"close"}]
```
deviceId设置ID，index第几个开关，status[open, close] 开关

- 返回
```
{success:true, msg:'操作成功'}
```

#### 调光`/north/control/{deviceId}/light` 
- 请求内容
```
{value:100}
```
deviceId设置ID，value 亮度

- 返回
```
{success:true, msg:'操作成功'}
```
